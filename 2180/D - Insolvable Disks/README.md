<div class="header"><div class="title">D. Insolvable Disks</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given $$$n$$$ different integer points $$$x_1 < x_2 < \ldots < x_n$$$ on the X-axis of the plane. For each $$$1 \le i \le n$$$, you have to pick a real value $$$r_i > 0$$$ and draw a disk with radius $$$r_i$$$ and center $$$x_i$$$ such that no two disks overlap.</p><p>You want to choose values $$$r_i$$$ in such a way that the number of tangent pairs of disks is maximized, and you have to find this maximum value.</p><p>We say that two disks overlap if the area of their intersection is positive. In particular, two outer tangent disks <span class="tex-font-style-bf">do not</span> overlap.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 2\cdot10^6$$$) — the number of points in that test case.</p><p>The second line of each test case contains $$$n$$$ integers $$$x_1, x_2, \ldots x_n$$$ ($$$0 \le x_1 < x_2 < \ldots < x_n \le 10^9$$$) — the coordinates of the points in that test case.</p><p>It's also guaranteed that the sum of $$$n$$$ over all test cases is at most $$$2\cdot10^6$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, print a
